# 📝 React Notes App

A clean and modern Notes application built using **React** and **Tailwind CSS**.  
Users can create and delete notes with a beautiful UI and responsive layout.

---

## 🚀 Features

- Add notes
- Delete notes
- Responsive layout
- Modern UI with Tailwind CSS
- Clean component structure

---

## 🛠 Tech Stack

- React
- Tailwind CSS
- JavaScript (ES6)
- Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/react-notes-app.git
```

Go into the project folder

```bash
cd react-notes-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
